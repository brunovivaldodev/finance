# Papucho

Papucho is a mobile app build with React Native Expo.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install foobar.
Install [expo](https://docs.expo.dev/get-started/installation/) first if you dont have it.

```bash
yarn install
```

## Run app

```bash
yarn start
```
